While learning full stack web development from Scrimba, I completed the HTML and CSS portion and then started JavaScript. This passenger counter was my first JavaScript project.

Key concepts learned and applied:

- Script tags — linking JavaScript to HTML
- Variables — storing and updating data
- Numbers — mathematical operations and incrementing
- Strings — concatenation to build saved entries
- console.log() — debugging and logging values
- Functions — writing reusable blocks of code
- DOM — manipulating the webpage with JavaScript
- getElementById — selecting HTML elements
- .innerText / .textContent — updating text on the page

Built using HTML, CSS and JavaScript.
